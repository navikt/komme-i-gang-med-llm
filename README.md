# Veileder for oppstart med språkmodeller i Nav

Dette prosjektet inneholder en veileder (skrevet i
[Quarto](https://quarto.org/)) som beskriver hvordan utviklere / produktteam i
Nav kan komme i gang med språkmodeller.

Veilederen inneholder beskrivelse av hvordan vi ivaretar personvern ved å
benytte [Veileder for
KI-prosjekter](https://data.ansatt.nav.no/quarto/db4222f2-f336-4fa4-81eb-219903894699/_book/index.html)
samt hvordan komme i gang rent teknisk på GCP med [Vertex
AI](https://cloud.google.com/vertex-ai?hl=nb).
